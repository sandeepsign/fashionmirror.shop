Integration Code

Add this script tag to your HTML:

<script src="http://localhost:5000/widget/mirror.js"></script>

Then add the button with data attributes:

<button
  class="mirror-me-button"
  data-merchant-key="mk_test_xxx"
  data-product-image="https://example.com/product.jpg"
  data-product-name="Product Name"
  data-product-price="99.99"
  data-product-currency="USD"
>
  Try It On
</button>

JavaScript API

For more control, use the JavaScript API:

const mirrorMe = MirrorMe.init({
  merchantKey: 'mk_test_xxx',
  theme: 'auto',
  onResult: (result) => {
    console.log('Try-on complete:', result);
  }
});

// Open programmatically
mirrorMe.open({
  product: {
    image: 'https://example.com/product.jpg',
    name: 'Product Name',
    price: 99.99
  }
});
